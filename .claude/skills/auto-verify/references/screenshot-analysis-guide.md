# Screenshot Analysis Guide

## What to Check

### 1. Layout Verification
- Elements positioned correctly (not overlapping)
- Proper spacing and alignment
- Responsive layout intact
- No elements cut off or hidden

### 2. Content Accuracy
- Correct text displayed
- Numbers/values correct
- Icons/images loading
- No placeholder text visible

### 3. State Consistency
- Buttons in correct state (enabled/disabled)
- Form fields showing right values
- Loading states resolved
- Error states shown when appropriate

### 4. Visual Regression
- Colors match expected theme
- Fonts rendering correctly
- Borders and shadows intact
- No unexpected visual changes

## Feature-Specific Checks

### Positions
- Position rows displaying correctly
- P&L values formatted and colored
- Exit/Add buttons visible
- Modal opens/closes properly

### Watchlist
- Instruments listed correctly
- Live prices updating (if visible)
- Add/Remove instrument works
- Search results display

### Option Chain
- CE/PE columns aligned
- Strike prices correct
- OI bars rendering
- Greeks displayed (if enabled)

### Strategy Builder
- Legs table correct
- P&L grid showing values
- Payoff chart rendering
- Add leg button works

### AutoPilot
- Dashboard cards showing data
- Strategy list populating
- Status badges correct
- Activity feed updating

## Red Flags
- Blank/white screen
- Spinner stuck loading
- Error message displayed
- Console errors in screenshot
- Missing data that should exist
- Broken images
